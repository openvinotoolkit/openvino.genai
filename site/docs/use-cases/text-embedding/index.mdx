# Semantic Search Using Text Embedding

Text embedding is now covered by the multimodal [`EmbeddingPipeline`](/docs/use-cases/multimodal-embedding), which computes embedding vectors for text, images, and videos with a single model. It also supports a text-only mode for dedicated text embedding models.

:::tip
See the [Semantic Search Using Multimodal Embedding](/docs/use-cases/multimodal-embedding) guide to get started.
:::
