## Run Model Using OpenVINO GenAI

OpenVINO GenAI supports the following video generation pipelines:
- [`Text2VideoPipeline`](https://docs.openvino.ai/2026/api/genai_api/_autosummary/openvino_genai.Text2VideoPipeline.html) for creating videos from text prompts.
- [`Image2VideoPipeline`](https://docs.openvino.ai/2026/api/genai_api/_autosummary/openvino_genai.Image2VideoPipeline.html) for creating videos conditioned on an input image and text prompt.
